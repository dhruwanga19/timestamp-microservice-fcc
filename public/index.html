Timestamp Microservice

This microservice provides a simple API to convert dates to Unix timestamps and UTC strings.

Example Usage:

Make a GET request to /api/:date? where :date is an optional date parameter.

• ISO date string: https://timestamp-gen-fcc.netlify.app/api/2015-12-25

• Unix timestamp: https://timestamp-gen-fcc.netlify.app/api/1451001600000

• Natural language date (e.g., "December 25, 2015") https://timestamp-gen-fcc.netlify.app/api/December 25, 2015

• Empty date parameter: https://timestamp-gen-fcc.netlify.app/api/ will return the current date

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
